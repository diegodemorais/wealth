## Module Refactoring Strategy (ES6 import/export)

### 01-preamble.mjs
- **Dependencies**: None
- **Exports**: `{ initFouc() }` — function that runs immediately
- **Side Effects**: Injects FOUC style into document.head
- **Change**: Wrap IIFE in exportable function

```javascript
export function initFouc() {
  const style = document.createElement('style');
  style.textContent = `...`;
  document.head.appendChild(style);
}
```

### 02-data-wiring.mjs
- **Dependencies**: window.DATA (injected inline in HTML before module import)
- **Challenge**: DATA is a placeholder string, injected post-template
- **Solution**: Export factory function that takes DATA parameter, calculates + exports all globals
- **Exports**: `{ initDataWiring(DATA) }` returns `{ CAMBIO, totalBrl, bucketUsd, geoUS, geoDM, geoEM, cagr, TWR_USD, TWR_BRL, today, progPct, _anoFireAlvoGlobal }`

```javascript
export function initDataWiring(data) {
  const CAMBIO = data.cambio;
  const PAT_GATILHO = data.premissas.patrimonio_gatilho;
  // ... calculations ...
  return { CAMBIO, totalBrl, bucketUsd, geoUS, geoDM, geoEM, cagr, TWR_USD, TWR_BRL, today, progPct, _anoFireAlvoGlobal };
}
```

## Bootstrap Module (bootstrap.mjs) — Orchestration

```javascript
// 1. FOUC guard runs immediately
import { initFouc } from './01-preamble.mjs';
initFouc();

// 2. Import all other modules (not executed yet)
import * as dataWiring from './02-data-wiring.mjs';
import * as utils from './03-utils.mjs';
import * as chartsPortfolio from './04-charts-portfolio.mjs';
import * as fireProj from './05-fire-projections.mjs';
import * as dashboardRender from './06-dashboard-render.mjs';
import * as initTabs from './07-init-tabs.mjs';

// 3. DATA is injected inline in HTML BEFORE this script loads
//    window.DATA = {...} is global

// 4. Initialize data wiring (computes derived values)
const dataDerived = dataWiring.initDataWiring(window.DATA);

// 5. Expose to window for cross-module use (legacy compatibility)
Object.assign(window, {
  // From 02
  CAMBIO: dataDerived.CAMBIO,
  totalBrl: dataDerived.totalBrl,
  // ... etc
  
  // From 03
  calcWellness: utils.calcWellness,
  fmtBrl: utils.fmtBrl,
  charts: utils.charts,
  
  // From 04-07
  buildTimeline: chartsPortfolio.buildTimeline,
  // ... etc
});

// 6. When DOM ready, run init
document.addEventListener('DOMContentLoaded', () => {
  initTabs.init();
});
```

## HTML Template Changes

Replace concatenated JS:
```html
<!-- OLD -->
<script>
  // 01-preamble + 02-data + 03-utils + ... + 07-init-tabs
</script>

<!-- NEW -->
<!-- DATA injection (inline) -->
<script>
  const DATA = {...}; // JSON string injected by build_dashboard.py
  const GENERATED_AT = new Date('...');
</script>

<!-- ES6 module bootstrap -->
<script type="module" src="bootstrap.mjs"></script>
```
