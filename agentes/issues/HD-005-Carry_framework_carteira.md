# HD-005: Carry Framework Aplicado a Toda a Carteira

## Status: Aberta
## Data: 2026-03-20
## Responsaveis: 02 Factor (lead), 08 Macro
## Prioridade: Baixa

## Contexto
Koijen mostrou no curso que "carry" (retorno se condicoes nao mudarem) e um framework unificador que se aplica a todas as classes de ativos: FX carry, bond carry, equity carry, commodity carry. Koijen, Moskowitz, Pedersen & Vrugt (2018) documentam que carry prediz retornos em todas as asset classes. Aplicar esse framework a carteira de Diego pode revelar insights sobre retorno esperado de cada posicao.

## Escopo
- Definir carry para cada bloco da carteira:
  - Equity (SWRD, AVGS, AVEM, JPGL): dividend yield + buyback yield
  - Renda+ 2065: taxa real (IPCA+ 7.01%) = carry
  - IPCA+ 2040: taxa real (IPCA+ 7.36%) = carry
  - HODL11: 0% carry (BTC nao gera yield)
  - BRL/USD: diferencial de juros (Selic - Fed Funds)
- Comparar carry entre blocos: onde esta o melhor "retorno sem fazer nada"?
- Avaliar se carry framework muda alguma prioridade de aporte
- Cross-asset momentum + carry: Koijen mostrou que combinar e superior a cada um isolado
