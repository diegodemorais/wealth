# FI-006: Intangibles e o Value Factor -- AVGS Esta Capturando o Value "Certo"?

## Status: Aberta
## Data: 2026-03-20
## Responsaveis: 02 Factor (lead)
## Prioridade: Media

## Contexto
O curso Chicago Booth (Koijen, PM Slides 2) mostra que medidas tradicionais de book-to-market nao capturam intangibles (R&D, software, brand value). "Intangible Value" (2021) propoe ajuste. Se intangibles nao sao capturados, empresas tech parecem "growth" mas podem ser "value" quando ajustadas -- e vice-versa. AVGS usa metodologia Avantis (value + profitability). Verificar se captura intangibles adequadamente.

## Escopo
- Entender metodologia Avantis para AVGS: como definem "value"? Usam book-to-market puro ou ajustado?
- Comparar com "Intangible Value" (2021): a metrica da Avantis captura intangibles?
- Se nao captura, qual o impacto? AVGS pode estar comprando "value traps" e ignorando "intangible value"
- Alternativas: existe ETF que use value ajustado por intangibles?

## Referencias do curso
- Koijen: PM Slides 2 (intangibles, "Intangible Value" 2021)
- Fama & French (2015): 5-factor model (profitability como proxy parcial)
