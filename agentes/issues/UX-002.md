## Executive Summary

**Current State: 0% Fidelity (DEV SERVER BROKEN)**

React dashboard v0.1.155 has **CRITICAL blocking issue**: dev server fails to start due to Tailwind v4 `@theme` syntax error in `globals.css`. The `@import "tailwindcss"` directive attempts to load `@tailwindcss/postcss` module which is not installed or properly configured.

**Rendered screenshots via Playwright show "Cannot GET /portfolio|/performance|/fire|/withdraw|/simulators|/backtest" errors — pages cannot be served until CSS build passes.**

### Root Cause Analysis

| Issue | Severity | Impact |
|-------|----------|--------|
| **`globals.css` imports Tailwind v4 syntax not installed** | 🔴 CRITICAL | Dev server crashes on load, all pages 404 |
| **PostCSS/Tailwind v4 migration incomplete** | 🔴 CRITICAL | Build succeeds (static export), dev server fails |
| **Previous shadcn/ui + Tailwind v4 migration left system in broken state** | 🔴 CRITICAL | Commits 6018e49, 2e1d94d, 0d11bc2 introduced bug |

### Path Forward

**Option A (Recommended): Revert to Tailwind v3 + Fix CSS Vars**
- Remove `@import "tailwindcss"` and `@theme { ... }` syntax
- Keep CSS variables defined in `dashboard.css` (working)
- Estimated effort: **45 min** (revert commit + test)
- Risk: Low (v3 is proven stable)

**Option B: Fix Tailwind v4 Setup (Higher Risk)**
- Install missing `@tailwindcss/postcss` package
- Update `postcss.config.js` for Tailwind v4
- Test full build pipeline
- Estimated effort: **90 min** (debugging + config)
- Risk: High (v4 has compatibility issues with existing codebase)

### BLOCKER-1: Tailwind v4 Import Error in globals.css

**File:** `/Users/diegodemorais/claude/code/wealth/react-app/src/app/globals.css` (line 1, 7-50)

**Current Code:**
```css
@import "tailwindcss";

@theme {
  --color-background: #0f172a;
  --color-foreground: #f1f5f9;
  /* ... 40+ theme tokens ... */
  --radius: 10px;
}
```

**Issue:**
- `@import "tailwindcss"` loads Tailwind v4 syntax
- Turbopack/PostCSS tries to find `@tailwindcss/postcss` module
- Module not in `package.json` or node_modules
- Dev server crashes on every page load

**Root Cause:** Commit 2e1d94d ("fix: Load Tailwind v4 via globals.css") introduced breaking syntax without installing dependencies.

**Recommended Fix (Option A):**
```css
/* REMOVE: @import "tailwindcss"; */
/* REMOVE: @theme { ... } block (all 40+ lines) */

/* Keep color palette as custom properties in new file: src/styles/tokens.css */
:root {
  /* Core backgrounds */
  --color-background: #0f172a;
  --color-foreground: #f1f5f9;
  
  /* Card / surface */
  --color-card: #1e293b;
  --color-card-foreground: #f1f5f9;
  
  /* ... rest of tokens ... */
}

/* Import existing dashboard CSS which already has all vars */
@import "./styles/dashboard.css";
```

**Effort:** 15 min (edit globals.css + restart server + verify)

**Test Command:**
